# AI-Powered-Blinkit-Business-Decision-Platform-

Blinkit Business Analysis Dashboard is a Streamlit-based data analytics project that analyzes customer data, marketing performance, and delivery efficiency using Python and Pandas. The dashboard provides interactive visualizations for marketing channel performance and customer feedback sentiment. A rule-based delay prediction module identifies delayed orders by comparing promised and actual delivery times. This project demonstrates practical data analysis and dashboard development skills using real-world datasets.
